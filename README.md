# IQ PLUS — Next.js 15 Premium Website

Stack: Next.js 15, React 19, Tailwind CSS 4, Motion for React, shadcn/ui primitives, Lucide Icons.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
Push the folder to GitHub, import the repository in Vercel, and deploy with the default Next.js settings.

## Pages
- `/` Home
- `/about` About
- `/courses` Courses
- `/gallery` Gallery
- `/reviews` Reviews
- `/contact` Contact

## Notes
- Arabic RTL throughout
- Registration dialog opens WhatsApp with pre-filled details
- Popup appears once per browser session
- No personal photos in testimonials
